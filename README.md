# Smart-Encrypted-AI-Attendance-System
Smart Encrypted AI Attendance System is a secure web-based attendance platform built with Django that uses face recognition and OTP verification to ensure authentic student attendance. The system protects sensitive data using encryption and provides separate student and admin dashboards for efficient management.
# Smart Encrypted AI Attendance System

A secure, web-based attendance management system built using **Django** that combines
**AI-powered face recognition**, **OTP verification**, and **data encryption** to ensure
authentic and tamper-proof student attendance.

maintaining data security and preventing proxy attendance.
